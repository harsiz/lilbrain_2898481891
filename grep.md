Grep is a thing in [[Bash]] terminals that let you search for text within a file.

think of it like a local `CTRL + F` but for files.

```bash
grep -R "tested" "/path/to/code"
```

`-R` checks for stuff recursively (so in a dir, and a dir of a dir etc)
